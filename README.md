# Mini SOC Brute-Force Detector

A beginner-friendly cybersecurity project that simulates a SOC analyst task:
analyzing authentication logs to detect potential brute-force attacks.

## Features
- Parses auth/SSH logs
- Detects failed login attempts
- Flags brute-force activity with severity levels
- Generates an incident-style report

## Tech
- Python
- Log analysis
- SOC-style alerting & reporting

## Usage
```bash
python src/bruteforce_detector.py
